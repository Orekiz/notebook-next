# shadecn/ui

提供一组由 Tailwind CSS 开发的 React 组件。

并非是组件库，因为他并不是传统组件库的那种 npm 包。只是将通过Tailwind CSS写好的组件，添加到项目中而已。

## 安装

### 手动

1. 添加 Tailwind CSS

因为是基于 Tailwind CSS 的，所以需要先 [安装 Tailwind CSS](https://tailwindcss.com/docs/installation)。

2. 添加依赖项

```shell
pnpm add tailwindcss-animate class-variance-authority clsx tailwind-merge
```

3. 添加图标库。

```shell
pnpm add lucide-react
```

4. 配置路径别名

组件中会使用路径别名来导入需要的包。

需要我们的项目配合来配置路径别名让其生效。

tsconfig.json

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "path": {
      "@/*": ["./*"]
    }
  }
}
```

7. 添加 cn 方法

组件中使用定义的 `cn` 方法来方便的添加 classname 。

lib/utils.ts

```ts
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

## 使用

支持通过 CLI 来添加组件。

但是有前提条件：

需要有 `components.json` 的配置文件。

创建配置文件：

```shell
pnpm dlx shadcn-ui@latest init
```

有了这个配置文件之后，就可以通过 CLI 来添加组件了。例如：

Alert 组件

```shell
pnpm dlx shadcn-ui@latest add alert
```
